## Usage
### Requirements
+ `Python` >= 3.8.10
+ `Pip` >= 20.0.2
+ **OS**: `Linux` 

### Install dependencies
``` sh
pip install -r requirements.txt
```

### Run program
``` sh
python3 CoffeeSystem.py
```

### Testing
``` sh
coverage run -m unittest TestCoffeeSystem
coverage report -m
coverage html
```

Watch `htmlcov/index.html`
